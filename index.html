分析不同范式的组织变革理论的局限性

分论点1：组织变革理论的局限性具有多层次差异

+

  • 先归纳已有局限，区分可修复程度不同的两类局限性。 在浅层与中层之外，提出第三类更顽固的局限性。

分论点2：理论的深层局限性具备不可修复性，其根源在于范式约束

+

正面论证：深层局限源于理论前提，而非理论细节。

  • P1
    任何理论都有其范式前提，理论细节在既定范式框架内展开论证。
    P2
    行动者网络理论的范式前提预设为：社会结构不是先在实体，而是网络互动结果。
    P3
    若某因素被界定为“结果”，则该因素不能同时作为同一解释链中的独立原因。
    C1
    因此，在行动者网络理论中，社会结构不能作为解释原因。
    P4
    现实中，组织变革常受结构与权力影响。
    C2
    行动者网络理论在组织变革中的结构解释方面存在局限性。
    C3
    这种局限性并非变量遗漏或论证疏漏，是由其范式前提造成。

横向比较论证：不同理论的优势与盲区互为映照。

  • P1
    行动者网络理论在范式前提中否认社会结构与权力的独立作用，其理论的深层局限表现为对社会结构与权力解释不足。
    P2
    激进结构主义在范式前提中强调社会结构与权力的决定性作用，其理论优势表现为能够解释社会结构与权力影响。
    P3
    若某理论的优势内容，恰好对应另一理论的盲区内容；且这种对应关系同步反映在双方前提上，则该优势与盲区并非偶然，而是由前提差异系统地产生。
    C
    范式前提影响了理论的深层局限性。

反面论证：若试图修复，也只能在原范式内打补丁。

  • P1
    理论修复通常发生在原有理论框架之内，而不改变其基本前提。
    P2
    功能主义三阶段模型将组织变革理解为系统调整，以实现组织与环境的适配。
    P3
    在该范式中，权力视角若被纳入，只能作为影响系统运行的变量，而不会影响解释框架本身。
    C1
    因此，该范式对“权力”视角的修复方式，只能是将其变量化并纳入原模型。
    P4
    激进结构主义将权力视为结构性压迫与矛盾来源，并可反思组织目标本身。
    P5
    修复后的功能主义范式理论不涉及对组织目标本身的否定。
    C2
    因此，即便是修复深层局限性，仍然没有改变理论原本的解释视角，只能在原体系内吸收新因素。


范式如何具体导致理论的深层局限

机制1：范式内部维度出于保持逻辑的一致性，形成了彼此依赖的关系，从而限制了解释路径的扩展

+

多个理论范式例子（功能主义 / 符号互动主义 / 激进结构主义）→ 提取共同结构 → 归纳出一般机制。

  • P1
    范式由多维构成，这些维度须在理论框架内保持逻辑一致性与整体自洽。
    P2
    核心维度的确立会对其他维度形成“兼容性约束”，不兼容选项被系统性排除。
    P3
    例如，基于本体论，认识论、方法论等对应匹配约束。
    C1
    理论的范式内部维度之间形成相互依赖的耦合结构。
    P4
    维度耦合形成“解释合法性条件”，规定什么是对象、证据与有效解释。
    C2
    范式内部维度的耦合预设了解释的合法空间，从而限制解释路径。

机制2：范式整体对研究问题的先验构造，即不同的范式把组织变革转变为范式可以解决的问题，从而形成了理论的解释边界

+

四个理论案例（功能主义 / 符号互动主义 / 激进结构主义 / 复杂性理论）→ 提取范式对现实的“重构”逻辑。

  • P1
    范式的各维度共同构成一套世界观，在资料收集之前决定现实如何被组织。
    P2
    不同范式会将同一现实重构为自身逻辑下“可解决的问题”。
    P3
    组织变革现象被分别转化为“适应/修复问题”“意义建构问题”“支配/意识形态问题”。
    C1
    范式在“解谜”之前先通过“问题化”，将复杂现象压缩为单一问题图景。
    P4
    组织变革现实具有复杂性。
    C2
    这种“问题化”机制牺牲现实的其他面向，形成理论解释边界。

核心结论

组织变革理论的解释边界，内在原因是其深层、不可修复的局限性，在本质上由其所依赖的范式结构所决定。其影响机制，具体是由范式内部多维度的耦合，以及范式对问题图景的构造二者作用下形成的。
